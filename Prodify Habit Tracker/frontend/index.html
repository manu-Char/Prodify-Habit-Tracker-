About Prodify

The application is built using a vibe coding approach, keeping the design clean, minimal, and user-friendly while maintaining practical functionality.

💡
Project Description

Prodify is a productivity web application designed to help users manage tasks and track daily habits. It features a clean dashboard that gives users a complete overview of their productivity at a glance.

  • Create and manage personal tasks
  • Build daily habits with one-completion-per-day logic
  • View productivity stats on the dashboard
  • Secure login with session-based authentication
